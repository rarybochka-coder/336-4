API та архітектура

Архітектура

Додаток побудований за трирівневою архітектурою:

РівеньТехнологіяОпис
КлієнтWindows FormsТонкий клієнт, викликає API
APIASP.NET CoreОбробляє запити, працює з БД
База данихSQLiteЗберігає всі дані

API Endpoints

МетодURLДія
GET/api/customersВсі клієнти
GET/api/customers/search?q=...Пошук
POST/api/customersДодати
DELETE/api/customers/{id}Видалити

Аналогічні endpoints існують для /api/orders, /api/items, /api/parts.

Запуск

API запускається на http://localhost:5000. Клієнт та API запускаються одночасно через Multiple Startup Projects у Visual Studio.
