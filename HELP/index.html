Довідкова система — Customer Orders App

Ласкаво просимо до довідки додатку управління замовленнями.

Примітка: Додаток є багатоланковим — клієнт (WinForms) взаємодіє з базою даних через Web API.

Зміст

  • Робота з клієнтами
  • Робота із замовленнями
  • Робота з позиціями
  • Робота з деталями
  • API та архітектура

Загальний опис

Додаток дозволяє керувати базою даних замовлень через зручний графічний інтерфейс. Основні можливості:

  • Перегляд та навігація по записах таблиць
  • Додавання нових записів
  • Видалення існуючих записів
  • Пошук по всіх полях таблиці
